# Lotto Number Recommendation Site

## Overview

This is a simple web application that generates and displays recommended lottery numbers.

## Features

*   **Lottery Number Generation:** Generates a set of unique random numbers within a specified range (e.g., 1-45).
*   **Display:** Displays the generated numbers to the user.
*   **User Interface:** A simple and intuitive interface with a button to trigger number generation.
